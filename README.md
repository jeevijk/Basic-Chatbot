# Basic Chatbot
This is a basic chatbot developed using Python.

## Description
The chatbot responds to user inputs with predefined replies.
It simulates a simple human-like conversation in the console.

## Features
* Interactive text-based conversation
* Predefined responses
* Easy to understand logic

## Technologies Used
 Python

## How to Run
1. Ensure Python is installed on your system
2. Run the chatbot using:
   python chatbot.py
